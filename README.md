# Kanban Board

Live site: [kanban-demo.live](https://kanban-demo.live)

A single-user Kanban task management app. Tasks are organized into Todo, Doing, and Done columns with full CRUD support.

## Tech Stack

- Nuxt 4
- Vue 3
- Pinia

Backend: [garyfan1/kanban_backend](https://github.com/garyfan1/kanban_backend)
